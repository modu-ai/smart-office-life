# AI 회의록 생성기

## Overview

회의 내용(텍스트, 메모, 음성 변환 텍스트)을 입력받아 한국 기업 표준 회의록 양식으로 자동 정리합니다.

## 회의록 표준 양식

```
                    회  의  록

회의명: ________________________________________________
일  시: ____년 __월 __일 (__요일) __:__ ~ __:__
장  소: ________________________________________________
작성자: ________________

[참석자]
| 소속 | 직급 | 성명 | 비고 |
|------|------|------|------|
|      |      |      | 주재 |
|      |      |      |      |

[안건]
1. ____________________________________________________
2. ____________________________________________________

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[안건 1] (안건 제목)

■ 논의 내용
  - (주요 발언 및 논의 사항 요약)
  - (참석자별 의견이 있을 경우 구분하여 기재)

■ 결정 사항
  - (합의된 결정 내용을 명확하게 기재)

■ 액션 아이템
| No | 할 일 | 담당자 | 기한 | 상태 |
|----|-------|--------|------|------|

[안건 2] (안건 제목)
  (동일 구조 반복)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[기타 논의사항]
  - (안건 외 논의된 사항)

[차기 회의]
  - 일시: ____년 __월 __일
  - 안건(예정): ________________________
```

## 회의록 작성 원칙

### 1. 결정사항 중심
- 논의 과정보다 **결정된 사항**을 먼저, 명확하게
- "~하기로 했다", "~로 결정되었다" 형태

### 2. 액션아이템 필수
- 모든 결정사항에는 반드시 **담당자 + 기한** 지정
- 상태: 예정 → 진행중 → 완료

### 3. 객관적 기술
- 주관적 해석 배제, 발언 내용을 객관적으로 기록
- 이견이 있었던 경우 양측 의견 모두 기재

### 4. 간결함
- 한 항목 2~3줄 이내로 요약
- 상세 자료는 "붙임" 또는 별도 문서로 참조

## 입력 유형별 처리

### Type 1: 메모/키워드 입력
사용자가 간단한 메모나 키워드를 제공하면 회의록 양식으로 확장합니다.

### Type 2: 음성 변환 텍스트 (STT)
음성인식으로 변환된 비정형 텍스트를 정제합니다:
- 화자 구분 (가능한 경우)
- 불필요한 추임새/반복 제거
- 핵심 내용 추출 및 구조화

### Type 3: 상세 내용 입력
상세한 회의 내용을 제공하면 양식에 맞춰 정리합니다.

## 부가 기능

### 액션아이템 트래커 (Excel)
회의에서 나온 액션아이템을 별도 Excel로 관리할 수 있습니다.

| 회의일 | 안건 | 액션아이템 | 담당자 | 기한 | 상태 | 완료일 | 비고 |
|--------|------|-----------|--------|------|------|--------|------|

### 회의 시리즈 관리
정기 회의의 경우 이전 회의 미완료 액션아이템을 자동 이관합니다.

## 출력 형식

- `.docx`: Word 문서 (기본값)
- `.hwpx`: 한글 문서
- `.xlsx`: 액션아이템 트래커
- `.md`: Markdown
- 텍스트: 간단한 회의 요약

## 워크플로우

```
회의 내용 입력 → 안건별 분류 → 결정사항/액션아이템 추출
→ 양식에 맞춰 정리 → 출력 형식 변환 → 전달
```
