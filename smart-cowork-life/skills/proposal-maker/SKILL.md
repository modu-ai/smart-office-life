# 제안서/견적서 생성기

## Overview

한국 기업 간 B2B 거래에서 필요한 제안서, 견적서, 계약서 초안을 전문적으로 생성합니다.

## 지원 문서 유형

| 문서 | 용도 | 출력 형식 |
|------|------|-----------|
| 사업 제안서 | 서비스/제품 도입 제안 | .docx, .pptx, .pdf |
| 견적서 | 가격/비용 제시 | .xlsx, .docx, .pdf |
| 입찰 제안서 | 공개/제한 입찰 참여 | .docx, .pdf |
| 계약서 초안 | 거래 조건 합의용 | .docx |

## 사업 제안서 구조

```
[표지]
- 제안서 제목
- 제출처: OOO 귀중
- 제출일: ____년 __월 __일
- 제출사: 회사명, 대표자명

[목차]

1. 제안 개요
   1.1 제안 배경 및 목적
   1.2 제안 범위
   1.3 기대 효과

2. 현황 분석
   2.1 귀사 현황 이해
   2.2 문제점 및 개선 기회
   2.3 벤치마킹 사례

3. 제안 솔루션
   3.1 솔루션 개요
   3.2 핵심 기능/서비스
   3.3 차별화 포인트
   3.4 적용 사례 (레퍼런스)

4. 추진 계획
   4.1 추진 체계 (조직도)
   4.2 단계별 일정
   4.3 투입 인력
   4.4 품질 관리 방안

5. 비용 제안
   5.1 비용 총괄표
   5.2 세부 비용 내역
   5.3 결제 조건

6. 회사 소개
   6.1 회사 개요
   6.2 주요 실적
   6.3 인증/수상 현황
   6.4 조직 현황

[붙임] 상세 견적서, 인력 프로필 등
```

## 제안서 작성 원칙

### 1. 고객 관점 우선
- "우리가 잘한다"보다 "귀사에 이렇게 도움이 된다"
- 문제 해결 중심의 스토리 라인

### 2. 숫자로 증명
- 정성적 표현보다 정량적 효과 제시
- "업무 효율 향상" → "처리 시간 40% 단축 (월 120시간 절감)"

### 3. 시각 자료 활용
- 다이어그램, 차트, 비교표 적극 활용
- 텍스트 : 시각자료 = 6 : 4 비율 권장

### 4. 경쟁사 대비 차별점
- 비교표를 통한 명확한 강점 제시
- 최소 3가지 핵심 차별점

## 출력 형식

- `.docx`: Word 문서 (기본값, 상세 제안서)
- `.pptx`: PowerPoint (프레젠테이션형 제안서)
- `.xlsx`: Excel (견적서, 비용표)
- `.pdf`: PDF (최종 제출용)

## 워크플로우

```
사용자 요청 → 문서 유형 파악 → 고객/제품 정보 수집
→ 구조 설계 → 섹션별 작성 → 견적표 생성
→ 검토/수정 → 출력 → 전달
```
