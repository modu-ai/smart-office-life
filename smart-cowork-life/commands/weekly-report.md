# 주간업무보고서 생성 커맨드

사용자가 간단히 업무 내용을 나열하면 한국식 주간보고서 양식으로 자동 정리합니다.

## 생성 양식

```
┌─────────────────────────────────────┐
│         주간업무보고서               │
├────────────┬────────────────────────┤
│ 작성자      │ (사용자명)             │
│ 보고 기간   │ YYYY.MM.DD ~ MM.DD    │
│ 작성일      │ YYYY.MM.DD            │
├────────────┴────────────────────────┤
│ ■ 금주 실적                         │
│   1. ...                            │
│   2. ...                            │
├─────────────────────────────────────┤
│ ■ 차주 계획                         │
│   1. ...                            │
│   2. ...                            │
├─────────────────────────────────────┤
│ ■ 이슈 및 건의사항                   │
│   - ...                             │
└─────────────────────────────────────┘
```

## 작성 규칙

1. 업무 항목은 **성과 중심**으로 작성 (예: "미팅 진행" → "고객사 미팅 3건 완료, 계약 2건 확정")
2. 금주 실적 / 차주 계획 / 이슈 3단 구조
3. 각 항목은 구체적 수치 포함 권장
4. 격식체(~합니다, ~하였습니다) 사용
5. korean-biz-docs 스킬의 보고서 양식 기준 적용

## 출력 형식

- 기본: 텍스트 (마크다운)
- 요청 시: .docx (Word 문서) / .hwpx (한글 문서)
