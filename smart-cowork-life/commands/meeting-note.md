# 회의록 빠른 생성 커맨드

사용자가 회의 내용을 자유 형식으로 입력하면 한국식 회의록 양식으로 자동 정리합니다.

## 생성 양식

```
┌─────────────────────────────────────┐
│              회 의 록                │
├────────────┬────────────────────────┤
│ 회의명      │ (자동 생성)            │
│ 일시        │ YYYY.MM.DD (요일) HH:MM│
│ 장소        │ (입력 또는 미기재)      │
│ 참석자      │ 홍길동, 김철수, ...     │
│ 작성자      │ (사용자명)             │
├────────────┴────────────────────────┤
│ ■ 안건                              │
│   1. ...                            │
├─────────────────────────────────────┤
│ ■ 논의 내용                          │
│   1. ...                            │
│   2. ...                            │
├─────────────────────────────────────┤
│ ■ 결정 사항                          │
│   1. ...                            │
├─────────────────────────────────────┤
│ ■ 액션아이템                         │
│   | 번호 | 내용 | 담당자 | 기한 |    │
│   |  1   | ...  | 홍길동 | MM/DD |   │
└─────────────────────────────────────┘
```

## 작성 규칙

1. 사용자 입력에서 **참석자, 안건, 결정사항, 액션아이템**을 자동 추출
2. 불명확한 항목은 "(확인 필요)"로 표시
3. 액션아이템은 반드시 담당자 + 기한 포함
4. meeting-minutes 스킬의 양식 기준 적용
5. 격식체(~하였음, ~합의함) 사용

## 출력 형식

- 기본: 텍스트 (마크다운)
- 요청 시: .docx (Word 문서) / .hwpx (한글 문서)
