# 슬기로운 코워크 생활

> 직장인·취준생을 위한 한국형 업무 스킬 플러그인

Claude Cowork에서 바로 사용할 수 있는 **12가지 업무 스킬**과 **5가지 슬래시 커맨드**를 제공합니다.

설치 방법은 [마켓플레이스 README](../README.md)를 참고하세요.

---

## 포함 스킬 (12종)

### 1. 비즈니스 이메일/공문 작성기

한국 비즈니스 이메일의 격식 수준 자동 조절, 제목 구조화, 한영 이메일 작성을 지원합니다.

- **지원 유형**: 업무 요청, 회신, 감사, 사과, 미팅 요청, 공지/안내, 거절, 영문 이메일
- **격식 3단계**: 최고 격식(임원/외부 대표) → 높은 격식(타 부서 팀장) → 보통 격식(동료)
- **제목 카테고리**: `[요청]` `[회신]` `[공유]` `[안내]` `[긴급]` `[확인]` `[보고]` `[FYI]`
- **CC/BCC 가이드**: 불필요한 CC 최소화, 의사결정권자 필수 포함 원칙
- **트리거 키워드**: 이메일, 메일, email, 회신, 답장, 공문

```
"거래처에 납품일정 변경 요청 이메일 써줘"
→ 격식 수준 자동 판별 → 제목 카테고리 적용 → 이메일 생성
```

### 2. AI 회의록 생성기

회의 내용을 한국식 회의록 양식(참석자, 안건, 결정사항, 액션아이템, 후속조치)으로 자동 정리합니다.

- **표준 양식**: 회의명, 일시, 장소, 참석자, 안건별 논의/결정/액션아이템
- **입력 유형**: 간단 메모/키워드, 음성 변환 텍스트(STT), 상세 내용
- **액션아이템 트래커**: 담당자+기한 필수 지정, 상태 추적(예정→진행중→완료)
- **회의 시리즈 관리**: 정기 회의 시 미완료 액션아이템 자동 이관
- **출력 형식**: `.docx`, `.hwpx`, `.xlsx`(액션아이템), `.md`
- **트리거 키워드**: 회의록, 회의, 미팅, 회의 정리, 액션아이템

```
"이번 주 회의 내용 정리해줘"
→ 안건별 분류 → 결정사항/액션아이템 추출 → 회의록 생성
```

### 3. 한국형 비즈니스 문서 생성기

한국 직장인이 매일 작성하는 문서를 자동 생성합니다. 기-승-전-결 구조, 결재선 포맷, 비즈니스 문체를 자동 적용합니다.

- **지원 문서**: 주간보고서, 업무일지, 기안서, 품의서, 지출결의서, 출장보고서, 회의록
- **결재선**: 담당→팀장→부장→이사 결재선 자동 생성
- **문체 규칙**: 격식체, 한 문장 40자 이내, 수치 중심 (추상적 표현 배제)
- **날짜 표기**: `2026. 02. 14.` / 문서번호: `HR-2026-0042`
- **출력 형식**: `.docx`, `.hwpx`, `.pdf`, `.md`
- **트리거 키워드**: 보고서, 기안서, 품의서, 지출결의서, 출장보고서, 결재

```
"이번 달 마케팅 예산 500만원 품의서 작성해줘"
→ 품의서 양식 → 결재선 + 품의 내역 테이블 → 문서 생성
```

### 4. 제안서/견적서 생성기

B2B 제안서, 견적서, 계약서 초안, 입찰 제안서를 전문적으로 생성합니다.

- **지원 문서**: 사업 제안서, 견적서, 입찰 제안서, 계약서 초안
- **제안서 구조**: 제안 개요→현황 분석→솔루션→추진 계획→비용 제안→회사 소개
- **견적서 양식**: 견적번호, 유효기간, 품목/수량/단가/금액, 부가세 자동 계산
- **작성 원칙**: 고객 관점 우선, 숫자로 증명, 시각 자료 활용, 경쟁사 대비 차별점
- **출력 형식**: `.docx`, `.pptx`, `.xlsx`(견적서), `.pdf`
- **트리거 키워드**: 제안서, 견적서, 입찰, RFP, 프로포절, proposal

```
"AI 컨설팅 서비스 제안서 만들어줘. 월 500만원, 3개월 프로젝트"
→ 제안서 구조 설계 → 섹션별 작성 + 견적표 생성
```

### 5. 엑셀 업무 자동화

매출 집계, 급여 계산, KPI 대시보드, 동적 배열 함수를 포함한 실무 엑셀 파일을 생성합니다.

- **템플릿**: 매출 집계표, 매출 분석 대시보드, 급여 계산서, 연차/휴가 관리, 거래처 관리, 미수금 관리, 재고 관리, 프로젝트 일정표, 경비 보고서
- **급여 계산**: 2026년 4대보험 요율 반영 (국민연금 4.75%, 건강보험 3.595%, 장기요양 13.14%, 고용보험 0.9%)
- **동적 배열 함수**: `XLOOKUP`, `FILTER`, `SORT`, `UNIQUE`, `TAKE`/`DROP`, `LET`
- **KPI 대시보드**: 3영역 분리(데이터→계산→대시보드), 조건부 서식, 5~7개 핵심 지표
- **한국 특화 수식**: 한국식 날짜, 금액 표기(원/천원), 전화번호 포맷, 부가세 포함 금액
- **트리거 키워드**: 엑셀, Excel, 스프레드시트, 매출 집계, 급여 계산

```
"영업팀 월별 매출 집계표 만들어줘"
→ 일별 매출 시트 + 월별 요약 시트 + 자동 수식 적용
```

### 6. 프롬프트 엔지니어링 마스터

AI 모델(Claude, ChatGPT 등)에게 최적의 결과를 이끌어내는 프롬프트 설계 5원칙과 26가지 기법을 제공합니다.

- **5원칙**: 명확성(Clarity), 맥락(Context), 구조(Structure), 제약(Constraints), 예시(Examples)
- **핵심 기법**: 제로샷, 퓨샷, 체인오브소트(CoT), 역할지정, 트리오브소트(ToT), 메타 프롬프트, 환각 방지
- **업무별 레시피**: 보고서 작성, 이메일 작성, 데이터 분석용 프롬프트 템플릿
- **프롬프트 체이닝**: 복잡한 작업을 여러 프롬프트로 분해하는 기법
- **플랫폼별 팁**: Claude, ChatGPT, Gemini, Midjourney/DALL-E 각각의 최적 활용법
- **트리거 키워드**: 프롬프트, prompt, AI 활용, 프롬프트 엔지니어링, CoT

```
"마케팅 보고서 작성에 최적화된 프롬프트 설계해줘"
→ 역할지정 + 구조화 + 제약 조건 적용 → 최적 프롬프트 생성
```

### 7. 프로젝트 관리 트래커

WBS, 간트차트, 진행률 대시보드, 이슈 트래커, RACI 매트릭스를 생성합니다.

- **WBS**: Level 0~3 계층 구조, 담당자/기간/선행작업/산출물/상태 추적
- **간트차트**: 텍스트 기반 간트 표현, 마일스톤 표기
- **진행률 대시보드**: 전체/단계별 진행률 막대그래프, 핵심 지표, 주간 진행 보고
- **이슈 트래커**: 이슈 ID, 우선순위(긴급/높음/보통/낮음), 카테고리, 진행 이력
- **RACI 매트릭스**: Responsible/Accountable/Consulted/Informed 역할 배분표
- **상태 아이콘**: ⏳대기, 🔄진행중, ✅완료, ⚠️지연, ❌취소, 🔒블로킹
- **트리거 키워드**: 프로젝트, WBS, 간트차트, 일정 관리, 마일스톤, PM

```
"신규 웹사이트 개발 프로젝트 WBS 만들어줘"
→ 기획→설계→개발→테스트→오픈 단계별 WBS + 간트 + RACI 생성
```

### 8. 포트폴리오·이력서 빌더

이력서, 자기소개서, 포트폴리오, 링크드인 프로필 최적화를 지원합니다. 2025~2026 채용 트렌드를 반영합니다.

- **이력서**: 한국형 표준 구조(기본정보/학력/경력/자격/기술역량), 수치화·동사 시작·STAR 원칙
- **자기소개서**: 4항목 구조(성장과정/지원동기/직무역량/입사포부), 항목별 500~1000자 가이드
- **포트폴리오**: 10페이지 이내 구조, STAR+ 프레임워크(Situation-Task-Action-Result+Plus)
- **링크드인**: 헤드라인 최적화(120자), About 섹션, 경력 STAR 기법, 네트워킹 체크리스트
- **AI 활용 주의**: AI 초안은 반드시 실제 경험으로 수정, 면접 대비 가능 여부 확인
- **트리거 키워드**: 이력서, 자기소개서, 포트폴리오, 링크드인, 취업, 이직

```
"마케팅 직무 이력서 검토해줘" / "링크드인 헤드라인 최적화해줘"
→ 수치화/STAR 기법 적용 여부 진단 → 개선 제안
```

### 9. 데이터 분석 리포트 생성기

엑셀/CSV 데이터를 분석하여 한국식 보고서 양식의 시각화 리포트를 자동 생성합니다.

- **분석 유형**: 매출 분석, KPI 대시보드, 비교 분석(전기/목표/경쟁사), 트렌드 분석, 고객 분석(RFM), 성과 보고서
- **1페이지 요약**: 핵심 KPI 4개 + 주요 인사이트 3줄 + 차트 + 액션 아이템
- **상세 보고서**: 분석 개요→현황→비교→심층 분석→인사이트 및 제언→부록
- **분석 워크플로우**: 데이터 탐색→정제→분석(pandas)→시각화→리포트 생성
- **숫자 표기법**: 억원 단위, 소수 1자리 %, ▲/▼ 증감 표시
- **출력 형식**: `.pptx`(경영진 보고), `.docx`, `.html`(인터랙티브), `.xlsx`, `.pdf`
- **트리거 키워드**: 데이터 분석, KPI, 대시보드, 매출 분석, 트렌드 분석

```
"이번 분기 매출 데이터 분석해서 경영진 보고서 만들어줘"
→ 데이터 탐색 → 기간별/품목별 분석 → KPI 대시보드 + 인사이트 리포트
```

### 10. 비즈니스 한영/영한 번역기

업종별 전문용어, 격식 수준, 문서 유형별 번역 스타일을 지원하는 비즈니스 특화 번역 스킬입니다.

- **번역 모드**: 이메일, 문서(보고서/제안서), 계약서, 마케팅(현지화), 회화(미팅/PT), 이력서
- **업종 전문용어**: IT/테크, 금융/회계, 법률/계약, 마케팅/광고, 인사/HR, 제조/물류 (7개 업종)
- **격식 매칭**: Formal↔격식체, Semi-formal↔정중체, Informal↔해요체
- **이메일 관용 표현**: 시작 인사, 요청/제안, 마무리 표현 한영 대조표
- **번역 원칙**: 의역 우선(자연스러움), 전문용어 일관성, 직급 호칭 변환
- **출력**: 텍스트, 대조표(.docx), 용어집(.xlsx)
- **트리거 키워드**: 번역, translate, 한영 번역, 영한 번역, 영작

```
"이 계약서를 영어로 번역해줘" / "이 영문 이메일을 한국어로 번역해줘"
→ 문서 유형 파악 → 격식 수준 결정 → 전문용어 매칭 → 번역
```

### 11. PPT 한국형 디자인 시스템

Pretendard+명조 타이포그래피, 6:3:1 색상 법칙, 한국 비즈니스 PPT 레이아웃 가이드라인을 적용합니다.

- **폰트 시스템**: Pretendard 9종(Thin~Black) + 조선일보명조(인용/격식용)
- **타이포그래피**: 제목 36~44pt ExtraBold, 소제목 24~28pt Bold, 본문 16~20pt Regular, KPI 48~72pt Black
- **색상 시스템**: "Midnight Executive" 기본 팔레트, 배경 60% + 텍스트 30% + 강조 10%
- **추가 팔레트**: Warm Corporate, Nature Green(ESG), Minimal Mono
- **레이아웃 패턴**: 표지, 섹션 구분, 2단 콘텐츠, 카드 그리드, 타임라인, KPI 대시보드
- **6×6 규칙**: 슬라이드당 글머리 최대 6개, 글머리당 단어 6개 이내
- **PptxGenJS 코드 컨벤션**: 표준 제목 바, 카드, 페이지 번호 헬퍼 함수 포함
- **트리거 키워드**: PPT 디자인, 슬라이드, 발표자료, 피치덱, PPT 템플릿

```
"AI 컨설팅 서비스 소개 PPT 10장 만들어줘"
→ 디자인 시스템 적용 → 표지+섹션+콘텐츠+KPI+마무리 슬라이드 생성
```

### 12. SVG 도식화 & Mermaid 변환기

Mermaid v11 + 커스텀 SVG로 비즈니스 문서 삽입용 고품질 다이어그램을 생성합니다.

- **Mermaid 지원**: 플로우차트, 시퀀스, 클래스, 상태, ER, 간트, 파이, 마인드맵, 타임라인, Git 그래프, 블록, 산키 (12종)
- **커스텀 SVG**: KPI 카드, 프로세스 화살표, 비교표(Before/After), 조직도, 수평 타임라인, 도넛 차트
- **v11 신기능**: `handDrawn` 외형(스케치 느낌), `ELK` 레이아웃 엔진
- **테마 색상**: ppt-design-system 연동, Pretendard 한글 폰트 자동 적용
- **SVG→PNG 변환**: sharp 라이브러리로 고해상도 PNG 변환 (PPT/Word 삽입용)
- **트리거 키워드**: 도식화, 다이어그램, SVG, 플로우차트, 조직도, mermaid

```
"프로젝트 승인 프로세스 플로우차트 만들어줘"
→ Mermaid 또는 커스텀 SVG 선택 → 테마 색상 적용 → SVG/PNG 생성
```

## 슬래시 커맨드 (5종)

### `/daily-report` — 일일업무보고서

오늘의 업무 내용을 간단히 입력하면 한국 직장 문화에 맞는 일일업무보고서를 자동 생성합니다.

- **구조**: 금일 업무 실적 → 진행 중 업무 → 익일 업무 계획 → 특이사항/건의사항
- **작성 규칙**: 경어체, 상태 표기(~완료/~진행중/~예정), 수치화 필수
- **출력**: 마크다운(기본), `.docx`, `.hwpx`

```
/daily-report 고객사 A 미팅 완료, 제안서 초안 50% 작성, 내일 팀 회의 예정
```

### `/weekly-report` — 주간업무보고서

이번 주 업무 내용을 나열하면 금주 실적/차주 계획/이슈 3단 구조의 주간보고서로 정리합니다.

- **구조**: 금주 실적 → 차주 계획 → 이슈 및 건의사항
- **작성 규칙**: 성과 중심, 구체적 수치, 격식체, korean-biz-docs 양식 기준
- **출력**: 마크다운(기본), `.docx`, `.hwpx`

```
/weekly-report 고객사 미팅 3건 완료, 제안서 초안 작성, 다음 주 최종 PT 예정
```

### `/quick-email` — 빠른 이메일 작성

상황만 설명하면 격식 수준을 자동 판별하여 비즈니스 이메일을 즉시 생성합니다.

- **톤 자동 판별**: 상급자/고객→격식체, 동료→정중체, 부하직원→부드러운 지시체
- **상황별 템플릿**: 요청, 회신, 사과, 공지 자동 적용
- **제목 규칙**: 핵심 내용 15자 이내, `[카테고리]` 접두사

```
/quick-email 거래처 부장님에게 납품 일정 1주 연기 요청
```

### `/meeting-note` — 빠른 회의록 정리

회의 내용을 자유 형식으로 입력하면 한국식 회의록 양식으로 자동 정리합니다.

- **자동 추출**: 참석자, 안건, 결정사항, 액션아이템을 입력에서 자동 분리
- **액션아이템**: 담당자+기한 필수, 불명확 항목은 "(확인 필요)" 표시
- **출력**: 마크다운(기본), `.docx`, `.hwpx`

```
/meeting-note 참석: 김과장, 이대리. 3월 프로젝트 착수 합의. 김과장이 WBS 작성 예정.
```

### `/resume-check` — 이력서·자기소개서 검토

이력서 또는 자기소개서를 분석하여 항목별 점수(1~5)와 구체적 개선 피드백을 제공합니다.

- **이력서 검토**: 수치화 여부, 액션 동사, STAR 기법, 키워드 매칭, 가독성
- **자기소개서 검토**: 글자수, 구체성, 회사 맞춤도, 중복 여부, 톤 일관성
- **개선 제안**: Before→After 형태로 수정 전/후를 보여줌

```
/resume-check (파일 첨부 또는 텍스트 직접 입력)
```
